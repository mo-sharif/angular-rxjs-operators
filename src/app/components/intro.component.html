RxJS & Observables in Angular

Data streams

A sequence of values over time

Examples

A number that goes up by one every second

[1, 2, 3, 4]

A stream of keypresses

["C", "o", "d", "e"]

A stream OF objects

[{{ '{ id: 1}' }},{{ ' { id: 2}' }}]

Reactive programming ?

We can define an application as a series of streams with operations that connect streams together and which are automatically called when new values are pushed onto those streams

Observables

An observable is essentially a stream (a stream of events, or data) and compared to a Promise, an Observable can be cancelled. It out of the box supports operators such as map() and filter().
